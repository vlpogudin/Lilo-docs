## Lilo Privacy Policy / Политика конфиденциальности

Last updated: February 10, 2026  
Дата обновления: 10 февраля 2026

This Privacy Policy describes how the Lilo mobile application collects, uses, stores, and deletes user data.  
Настоящая Политика конфиденциальности описывает, как мобильное приложение Lilo собирает, использует, хранит и удаляет данные пользователей.

App name / Название приложения: Lilo  
Android package / Android package: com.lilo.app

---

# 1. Data We Collect / Какие данные мы собираем

When you use the application, we may collect and process the following categories of data:

- account data: email address, authentication identifiers (Firebase Auth UID), and display name;
- profile data: name and date of birth, if voluntarily provided by the user during registration or profile setup;
- user content: notes, quotes, favorites, bookmarks, and in-app reading progress;
- technical notification data: Firebase Cloud Messaging token and limited device metadata required to deliver push notifications.

We do not collect health records, medical diagnoses, or medical treatment information.

При использовании приложения мы можем собирать и обрабатывать следующие категории данных:

- данные аккаунта: адрес электронной почты, идентификаторы авторизации (Firebase Auth UID) и отображаемое имя;
- данные профиля: имя и дата рождения, если пользователь добровольно указывает их при регистрации или настройке профиля;
- пользовательский контент: заметки, цитаты, избранное, закладки и прогресс чтения внутри приложения;
- технические данные уведомлений: токен Firebase Cloud Messaging и ограниченные технические данные устройства, необходимые для доставки push-уведомлений.

Мы не собираем медицинские записи, диагнозы или информацию о лечении.

---

# 2. Why We Use Data / Зачем мы используем данные

We use collected data for the following purposes:

- account management: creating and maintaining user accounts;
- app functionality: enabling core features such as reading sessions, notes, bookmarks, quotes, and progress tracking;
- notifications: delivering push notifications related to app activity;
- security and reliability: preventing abuse, maintaining system integrity, and improving service stability.

Мы используем собранные данные для следующих целей:

- управление аккаунтом: создание и поддержка учетных записей пользователей;
- работа функций приложения: обеспечение основных функций, таких как чтение, заметки, закладки, цитаты и отслеживание прогресса;
- уведомления: отправка push-уведомлений, связанных с работой приложения;
- безопасность и надежность: предотвращение злоупотреблений, поддержание целостности системы и повышение стабильности сервиса.

# 3. Legal Basis for Processing / Правовые основания обработки

Where applicable under data protection laws, we process personal data based on the following legal grounds:

- user consent – when the user voluntarily provides information;
- legitimate interests – such as improving service reliability and security.

В случаях, предусмотренных законодательством о защите данных, обработка персональных данных осуществляется на следующих основаниях:

- согласие пользователя – когда пользователь добровольно предоставляет данные;
- законные интересы – например улучшение надежности и безопасности сервиса.

---

# 4. Data Storage and Providers / Хранение данных и провайдеры

The application uses Google Firebase services, including:

- Firebase Authentication;
- Cloud Firestore;
- Firebase Cloud Messaging.

User data may be processed and stored on servers operated by Google or its infrastructure providers. Processing is subject to Google's data protection and security policies.

Приложение использует сервисы Google Firebase, включая:

- Firebase Authentication;
- Cloud Firestore;
- Firebase Cloud Messaging.

Данные пользователей могут обрабатываться и храниться на серверах, управляемых компанией Google или ее инфраструктурными провайдерами. Обработка данных осуществляется в соответствии с политиками безопасности и защиты данных Google.

---

# 5. Data Sharing / Передача данных третьим лицам

We do not sell, rent, or trade personal data.

Data may be shared only with trusted service providers necessary to operate the application, such as Google Firebase. These providers process data solely for the purpose of delivering the services used by the application.

Мы не продаем, не сдаем в аренду и не передаем персональные данные в коммерческих целях.

Передача данных возможна только доверенным сервис-провайдерам, необходимым для работы приложения, например Google Firebase. Эти провайдеры обрабатывают данные исключительно для обеспечения работы используемых сервисов.

---

# 6. Data Retention / Сроки хранения данных

We retain personal data for as long as:

- your account remains active;
- it is necessary to provide the service.

When the account is deleted, associated personal data is removed from active systems.

Мы храним персональные данные:

- пока ваш аккаунт активен;
- пока это необходимо для предоставления сервиса.

При удалении аккаунта связанные персональные данные удаляются из активных систем приложения.

---

# 7. Account and Data Deletion / Удаление аккаунта и данных

You can delete your account directly within the application:

- open Profile settings;
- tap “Удалить аккаунт”;
- confirm deletion.

After confirmation, the account and associated personal data will be removed from the application's active databases.

If deletion fails due to security restrictions (for example, recent login required), please sign in again and repeat the deletion process.

Вы можете удалить аккаунт непосредственно в приложении:

- откройте настройки профиля;
- нажмите «Удалить аккаунт»;
- подтвердите удаление.

После подтверждения аккаунт и связанные персональные данные удаляются из активных баз данных приложения.

Если удаление невозможно из-за требований безопасности (например, recent login required), выполните повторный вход в аккаунт и повторите удаление.

# 8. International Data Transfers / Международная передача данных

Because the application uses cloud infrastructure providers, user data may be processed or stored in countries other than the user's country of residence. Such transfers are performed in accordance with applicable data protection laws.

Поскольку приложение использует облачную инфраструктуру, данные пользователей могут обрабатываться или храниться в странах, отличных от страны проживания пользователя. Такая передача осуществляется в соответствии с применимыми законами о защите данных.

# 9. Children / Дети

The application is not intended for unlawful use by children where parental consent is required by local law. If we become aware that personal data of a child has been collected without appropriate consent, we will take steps to delete such data.

Приложение не предназначено для использования детьми в случаях, когда по местному законодательству требуется согласие родителей. Если нам станет известно, что персональные данные ребенка были собраны без соответствующего согласия, мы примем меры для удаления таких данных.

Мы используем стандартные технические и организационные меры для защиты персональных данных. Однако ни один способ передачи данных через Интернет или их хранения не является полностью безопасным.
